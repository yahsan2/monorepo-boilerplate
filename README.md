# Monorepo boilerplate


## Setup

```bash
# npm
npm install
```

## Development Server
Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

```

```bash
# only frontend
npm run @web dev
```

```bash
# only storybook
npm run @ui dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```
